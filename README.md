# Student Performance Prediction

## Overview
This project predicts the final grade (G3) of students using Machine Learning.

## Technologies Used
- Python
- Scikit-Learn
- Streamlit
- Plotly
- Pandas
- NumPy

## Features
- Predict Final Grade
- Performance Analysis
- Pass/Fail Status
- Interactive Dashboard
- Charts and Visualization

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
